# Project 6: Python Lists and Tuples

This project contains a Python script that demonstrates the use of lists and tuples in Python. It covers creating lists, accessing elements, modifying lists, sorting, and various list methods. It also briefly touches on tuples and their immutability.

## Files
- `lists.py`: The Python script.

To run the script, use the following command:

python project06/lists.py
