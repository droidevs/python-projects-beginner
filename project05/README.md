# Project 5: Rock, Paper, Scissors

This project contains a simple, command-line Rock, Paper, Scissors game. The user is prompted to choose between rock, paper, or scissors, and the computer makes a random choice. The winner is then announced.

## Files
- `rps.py`: The Python script for the game.

To run the script, use the following command:
```bash
python project05/rps.py
```
