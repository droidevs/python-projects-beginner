# Project 7: Python Dictionaries and Sets

This project contains a Python script that demonstrates the use of dictionaries and sets in Python. It covers creating dictionaries, accessing and modifying data, and various dictionary methods. It also introduces sets and their properties, including how to perform set operations like union and intersection.

## Files
- `dictionaries.py`: The Python script.

To run the script, use the following command:
```bash
python project07/dictionaries.py
```
