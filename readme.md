# Python Mini-Projects

This repository contains a collection of small Python projects, each designed to demonstrate a specific concept or feature of the Python language.

---

## Project Summaries

### Project 1: Hello World
- **Description:** A simple "Hello World!" script.
- **Files:** `hello.py`

### Project 2: Welcome
- **Description:** Prints a welcome message to the console.
- **Files:** `welcome.py`

### Project 3: The Meaning of Life
- **Description:** Demonstrates the use of a ternary operator.
- **Files:** `meaning.py`

### Project 4: Python Data Types
- **Description:** Demonstrates various data types in Python.
- **Files:** `data_types.py`

### Project 5: Rock, Paper, Scissors
- **Description:** A simple command-line Rock, Paper, Scissors game.
- **Files:** `rps.py`

### Project 6: Python Lists and Tuples
- **Description:** Demonstrates the use of lists and tuples.
- **Files:** `lists.py`

### Project 7: Python Dictionaries and Sets
- **Description:** Demonstrates the use of dictionaries and sets.
- **Files:** `dictionaries.py`

### Project 8: Loops and RPS V2
- **Description:** Demonstrates loops and an improved Rock, Paper, Scissors game.
- **Files:** `loops.py`, `rps2.py`

### Project 9: Functions in Python
- **Description:** Demonstrates how to define and use functions.
- **Files:** `functions.py`

### Project 10: Recursion, Loops, and RPS v3
- **Description:** Demonstrates recursion and a recursive Rock, Paper, Scissors game.
- **Files:** `recursion.py`, `example.py`, `rps3.py`

### Project 11: Scope and RPS v4
- **Description:** Demonstrates variable scope and an updated Rock, Paper, Scissors game.
- **Files:** `scope.py`, `rps4.py`

### Project 12: Closures and Rock, Paper, Scissors V5
- **Description:** Explores closures and a Rock, Paper, Scissors game using closures.
- **Files:** `closure.py`, `rps5.py`

### Project 13: F-Strings and Rock, Paper, Scissors V6
- **Description:** Demonstrates f-strings and a Rock, Paper, Scissors game using f-strings.
- **Files:** `f_strings.py`, `rps6.py`

### Project 14: Modules and Rock, Paper, Scissors V7
- **Description:** Demonstrates the use of modules and a modular Rock, Paper, Scissors game.
- **Files:** `modules.py`, `kansas.py`, `rps7.py`

### Project 15: Command-Line Arguments and Rock, Paper, Scissors V8
- **Description:** Demonstrates `argparse` for command-line arguments in a personal greeting script and the Rock, Paper, Scissors game.
- **Files:** `hello_person.py`, `rps8.py`

### Project 16: The Arcade
- **Description:** A mini arcade with two games: Rock, Paper, Scissors, and Guess My Number.
- **Files:** `arcade.py`, `guess_number.py`, `rps.py`

### Project 17: Lambda Functions
- **Description:** Demonstrates the use of lambda functions.
- **Files:** `lambda.py`

### Project 18: Classes and Object-Oriented Programming (OOP)
- **Description:** An introduction to OOP with classes, inheritance, and polymorphism.
- **Files:** `classes.py`

### Project 19: Exception Handling
- **Description:** Demonstrates exception handling with `try`, `except`, `else`, and `finally`.
- **Files:** `exceptions.py`

### Project 20: Object-Oriented Programming (OOP) Banking System
- **Description:** A practical OOP project simulating a basic banking system.
- **Files:** `bank_accounts.py`, `oop_project.py`

### Project 21: Weather App
- **Description:** A command-line app to fetch current weather conditions using the OpenWeatherMap API.
- **Files:** `weather.py`

### Project 22: File Handling in Python
- **Description:** A comprehensive demonstration of file handling operations.
- **Files:** `files.py`

### Project 23: Flask Weather Web Application
- **Description:** A web application built with Flask to retrieve and display weather conditions.
- **Files:** `server.py`, `weather.py`
